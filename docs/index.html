Screenshotty screenshot API

Capture pixel-perfect screenshots and PDFs of any website or raw HTML with a single, fully-typed call — powered by Screenshotty.

Get a free API key View on GitHub

Quick start

import { Screenshotty } from "screenshotty-js";

const client = new Screenshotty(process.env.SCREENSHOTTY_API_KEY);

// Raw PNG bytes
const png = await client.capture({ url: "https://example.com", fullPage: true });

// …or just the hosted URL
const url = await client.captureToUrl({ url: "https://example.com" });

Grab a free key (1,500 screenshots/month, no credit card) from the Screenshotty dashboard, or try requests live in the interactive playground.

What you can capture

  • Full-page & elementWhole scrollable pages or a single CSS selector.
  • Any deviceViewport presets from desktop 4K to iPhone, with device scale.
  • PNG, JPEG, WebP, PDFOne format option, including PDF export.
  • Clean shotsAd-blocking and cookie-banner removal built in.
  • Dark mode & geoForce color scheme; target by country.
  • Auth & injectionCookies, headers, and custom JS/CSS before capture.

Website → PDF

const pdf = await client.capture({
  url: "https://example.com",
  format: "application/pdf",
  printed: true,
});
